## CodSoft AI Internship Projects

## Task 1: Rule-Based AI Chatbot

# Project Overview
This project is a Rule-Based AI Chatbot developed using Python for the CodSoft AI Internship.
The chatbot interacts with users through text input and provides predefined responses based on user queries.

# Features
- User name greeting
- Basic conversation
- Answers questions about Python and Artificial Intelligence
- Help command to view available queries
- Exit command to close the chatbot

# Technologies Used
- Python

# How to Run
1. Download or clone this repository.
2. Open terminal in the project folder.
3. Run the command:
      python chatbot.py
